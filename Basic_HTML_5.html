This paragraph shows the ampersand (&), along with "double quotation marks". In HTML these characters are reserved characters and entities are used to display them. So the browser doesn't mix up or confuse the characters HTML uses "entities". Reserved characters can be displayed using an entity name (&entity_name;) or an entity number (&#entity_number;). These characters £, ¥, € have been displayed using entity names and these characters £, ¥, € with the entity numbers.
